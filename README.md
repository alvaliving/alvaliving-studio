# Alvaliving Studio

**Brand Strategy & UX Web Design. Building digital experiences for brands, businesses, and institutions. Where creativity meets strategic clarity.**

This repository contains public assets, plugins, documentation, and technical guides developed by Alvaliving Design Studio.

## Services

- Responsive web design and UX optimization  
- SEO strategy and schema implementation  
- Legal compliance (GDPR, cookie banners, privacy policies)  
- Email deliverability (SPF, DKIM, DMARC, SMTP validation)  
- WordPress development and modular MU-plugins  
- Technical documentation and client onboarding assets  
- Branding and visual identity for digital platforms  

## Featured Projects

- `mu-plugins/` → WordPress snippets (login branding, SVG support, admin tweaks)  
- `checklists/` → DNS, email, and compliance validation workflows  
- `assets/` → Logos, icons, and visual components for client use  
- `public/` → HTML/CSS/JS boilerplates for landing pages and microsites
